# Paketo Tiny Stack

## Use Case
Ideal for:
- Most Go apps
- Java [GraalVM Native Images](https://www.graalvm.org/docs/reference-manual/native-image/)

Contains:
- Build: ubuntu:bionic + openssl + CA certs + compilers + shell utilities
- Run: distroless-like bionic + glibc + openssl + CA certs
